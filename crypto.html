Crypto CTF Tool

This is a tool for optimizing the solving of crypto ctf challenges. It includes a solver for simple ciphers like rot13 and bubblebabble, decoders for various bases, tools for splicing and combining input, and functions for specific tasks like brute forcing zip files. It also contains an auto solver that will brute force various combinations of decoders and ciphers in an attempt to solve a challenge.

The script will print the help menu on start, which provides a summary of the avaliable commands. 


$ ./tool.py
Inputs:	input, read, update
 Ciphers:	rot13, xor, shift, sub, bubblebabble, baconian
 Decoders:	hex, bases, base64, base32, base16
 Mod:		remove, getupper, toupper, tolower
 Replace:	upper, lower, symbols, newlines, spaces
 Zip:		zip brute
 Info:		length
 Utility:	system, history, brute, clear, exit
>>
Let's solve a few example challenges. The first is a simple one: we are given the string SYNTPrfneVfPbbyOhgAbgFrpher

Looks like a simple rotation, so let's add the string to the queue and apply the rot13 cipher. 


>> input SYNTPrfneVfPbbyOhgAbgFrpher
SYNTPrfneVfPbbyOhgAbgFrpher
>> rot13
FLAGCesarIsCoolButNotSecure
>>
Looks like we have our flag.

Let's try another challenge. This time the string is xipak-comok-repuk-vanik-dytuk-dimyk-sinyx. This looks like a format I've seen before, but I can't remember where. Let's try the the brute command which will automatically try various ciphers. 


>> brute
Form is: FLAG
One time decodes:
hackingbubble
>>
Looks like it recognized the bubblebabble cipher automatically.

These are just two examples. The script contains many more features described in the help menu.
